# EXP-008: does the delta advantage survive 50M?

- Hypothesis served: H2/H6 scaling — one honest 18M -> 50M trend point for the
  report's central claim
- Date: pre-registered 2026-07-05, before any results
- Status: CONFIGURED (feasibility-gated)

## Design

V3-50M (delta, d768/8L/delta_every2/window128/d_k=d_v=504, 50.52M) vs B2-50M
(tf_pp, d768/8L/d_ff2048, 50.54M; match -0.05%; d_k must divide n_heads=12 —
504/12=42, the 512 first draft crashed the probe). Families: algo_exec
(computation, tightest variance) + state_guard (memory; compress excluded on
VRAM projection). 4000 steps, lr 4.2e-4 (depth-scaled for 8L), seeds 0-1.
Note: d_k/d_model = 0.66 at 50M vs 0.75 at 18M (kept for param match; a
confound only if the verdict is borderline, flagged here in advance).

## Feasibility gate (probe BEFORE the run)

200-step V3-50M probe on algo_exec + 100-step on state_guard:
- throughput >= 5k tok/s AND peak VRAM < 30GB -> GO
- else -> SKIP permanently (no kernel-engineering detour; one sentence in the
  report's limitations). Probe cost ~$1.

**Probe result (2026-07-05): GO.** algo_exec 8816 tok/s / 10.2GB peak;
state_guard 10059 tok/s / 21.7GB peak. (First probe attempt crashed: d_k=512
does not divide n_heads=12; fixed to d_k=504, which also tightened the param
match to -0.05%.) algo_exec V3 jobs run 2-wide; state_guard solo.

## Pre-registered read

Reference gaps at 18M (all-tier, 3 seeds): algo_exec +21.8, state_guard +4.8.

1. Ceiling guard: if V3-50M all-tier > .95 on a family, primary read moves to
   tiers 3-5 for that family (EXP-007 lesson).
2. Retention: gap(50M) / gap(18M) per family. >= 0.5 on both -> "advantage
   scales"; >= 0.5 on one -> mixed; < 0.5 on both -> "advantage is
   small-scale-concentrated" (an honest slope for the report either way — this
   experiment has no kill condition, only a characterization).
3. flops/correct comparison at 50M (does the efficiency edge persist?).
4. 2 seeds: no stability gate at this arm size; seed range reported verbatim.

## Cost

Probe ~$1; full run 8 jobs, V3 jobs solo (~3.5h each, sequential) + B2 jobs
fast -> ~15h wall, ~$11 at $0.69/hr. Hard cap 20h.
